# \# ⚡ Ultimate QR Toolkit (Chrome Extension)

# 

# A lightning-fast, 2-in-1 Chrome extension that instantly scans QR codes directly from your screen and generates QR codes for the websites you visit. 

# 

# No more pulling out your phone to scan your computer screen. 

# 

# \## ✨ Key Features

# \* \*\*Lightning-Fast Screen Scanner:\*\* Instantly finds and reads any QR code visible on your screen using an optimized 50% downscaling speed hack.

# \* \*\*Smart QR Generator:\*\* If no QR code is found on the screen, it automatically generates a high-quality QR code for the current tab you are viewing.

# \* \*\*Quick Actions:\*\* One-click buttons to instantly copy the current page link or download the generated QR code as an image file.

# \* \*\*History Vault:\*\* Automatically saves your 5 most recently scanned links so you never lose them. (Can be toggled off or cleared instantly).

# \* \*\*Right-Click Scanning:\*\* Adds a convenient "Scan this QR Code" option to your right-click context menu for images.

# \* \*\*100% Local \& Private:\*\* All scanning, generating, and history storage happens directly inside your browser. No data is sent to external servers.

# \## 🛠️ Installation Guide (Developer Mode)

# 

# Since this extension is not currently published on the Chrome Web Store, you will need to install it manually using Developer Mode. It takes less than 60 seconds.

# 

# \### For Google Chrome:

# 1\. Download the ZIP file of this project and \*\*extract/unzip\*\* the folder to a safe place on your computer.

# 2\. Open Google Chrome and type `chrome://extensions/` into the URL bar and hit Enter.

# 3\. In the top right corner, turn on the \*\*Developer mode\*\* toggle switch.

# 4\. Click the \*\*Load unpacked\*\* button that appears in the top left.

# 5\. Select the extracted folder (make sure you select the folder that contains the `manifest.json` file).

# 6\. \*\*Done!\*\* Pin the extension to your toolbar by clicking the puzzle piece icon in the top right of Chrome.

# 

# \*Note: Chrome may occasionally show a popup asking you to "Disable developer mode extensions." This is a standard Chrome security warning for unpackaged extensions. Simply click "Cancel" or "X" to keep using the toolkit.\*

# 

# \### For Microsoft Edge (Alternative):

# 1\. Extract the downloaded ZIP folder.

# 2\. Open Edge and type `edge://extensions/` into the URL bar.

# 3\. Turn on the \*\*Developer mode\*\* toggle in the bottom left corner.

# 4\. Click \*\*Load unpacked\*\* and select the extracted folder.

# \## 🚀 How to Use

# \* \*\*To Scan:\*\* Make sure the QR code is visible somewhere on your screen, then click the extension icon in your toolbar.

# \* \*\*To Generate:\*\* Click the extension icon on any webpage that \*does not\* have a QR code on it. It will instantly create one for you.

# \* \*\*To Access History:\*\* Open the extension and look at the "Recent Scans" list at the bottom.
